# UI Style Guide

> This is a sample style guide for the UILint test application.

## Colors

- **Primary**: #3B82F6 (used in buttons, links, accents)
- **Secondary**: #6B7280 (used in muted text, secondary actions)
- **Success**: #10B981 (used in success states)
- **Warning**: #F59E0B (used in warning states)
- **Error**: #EF4444 (used in error states)
- **Background**: #FFFFFF, #F3F4F6, #F9FAFB
- **Text**: #111827 (headings), #374151 (body), #6B7280 (muted)

## Typography

- **Font Family**: system-ui, -apple-system, sans-serif
- **Font Sizes**: 12px, 14px, 16px, 18px, 20px, 24px, 32px, 48px
- **Font Weights**: 400 (normal), 500 (medium), 600 (semibold), 700 (bold)
- **Line Heights**: 1.25 (headings), 1.5 (body), 1.75 (relaxed)

## Spacing

- **Base unit**: 4px (0.25rem)
- **Common values**: 4px, 8px, 12px, 16px, 20px, 24px, 32px, 48px, 64px
- **Component padding**: 16px (small), 24px (medium), 32px (large)

## Border Radius

- **Small**: 4px (rounded)
- **Medium**: 8px (rounded-lg)
- **Large**: 12px (rounded-xl)
- **Full**: 9999px (rounded-full)

## Components

- **Buttons**: rounded-lg, px-4 py-2, font-medium, transition-colors
- **Cards**: rounded-xl, shadow-sm, p-6, bg-white
- **Inputs**: rounded-lg, border border-gray-300, px-3 py-2
- **Badges**: rounded-full, px-2 py-1, text-sm font-medium
